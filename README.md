# Практическая работа №5

Заметки

На оценку 4:

- создать приложение на Flutter, которое будет представлять собой заметки. 

- пользователь сможет вводить текст в поле, и при нажатии на кнопку "Сохранить" введённый текст будет отображаться ниже в виде списка.
